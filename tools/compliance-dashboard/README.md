# Compliance Command Center

A Proof of Concept (PoC) dashboard for visualizing Office 365 Compliance and Security data.

## Features

- **Overview**: High-level metrics and system health.
- **DLP & Audit**: Visualizations of audit logs and sensitivity labels.
- **Access Control**: Simulated Conditional Access policy management.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run locally:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```
